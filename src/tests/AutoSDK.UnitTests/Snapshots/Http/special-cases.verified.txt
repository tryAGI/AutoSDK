### List all pets
# @name listPets
GET {{host}}/pets?limit={{limit}}

### Create a pet
# @name createPets
POST {{host}}/pets
Accept: application/json

### Info for a specific pet
# @name showPetById
GET {{host}}/pets/{{petId}}
Accept: application/json
